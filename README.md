# Clawdbot IQ Test

**Verifiable intelligence benchmark for Clawdbot agents.**

## What is this?

A standardized test suite for measuring Clawdbot capabilities:
- **Memory** - Cross-session recall, search accuracy, update correctness
- **Reasoning** - Multi-step planning, novel problem solving
- **Tools** - Correct selection and efficient usage
- **Context** - Proper file awareness and management

## Why?

The community at [moltbook.com](https://www.moltbook.com/) is experimenting with memory systems and intelligence improvements, but there's no way to **verify** what actually works. This benchmark provides:
